# Sistema de Conta Bancária em Java

Projeto simples desenvolvido em Java para praticar conceitos de Programação Orientada a Objetos (POO).

## Funcionalidades

- Abertura de conta (Conta Corrente ou Conta Poupança)
- Depósito de dinheiro
- Saque de dinheiro
- Pagamento de mensalidade da conta
- Exibição do estado atual da conta

## Conceitos aplicados

- Programação Orientada a Objetos (POO)
- Encapsulamento
- Getters e Setters
- Construtores
- Métodos de negócio

## Tecnologias utilizadas

- Java
